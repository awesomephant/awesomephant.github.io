Projects based on research I've done so far

Explore the social implications of machine learning

Machine learning applications often allow you to make guesses on the sort of people who developed them and the sort of data that was used to train them.

One example of this is facial recognition software not detecting black faces because the engineers building the software are overwhelmingly white. There are earlier examples of this codified racism, for example black skin not showing up on film correctly because they were only tested with white models.

Do something more interesting with tweets

I did a project a few years ago where I collected every tweet containing the words "Good Morning" (in a number of different languages) over a 24-hour period. Then I mapped each tweet based on the time it was posted and its geographic location, so you could see the rotation of the earth represented.

I like the idea of finding representations of basic natural processes in places where you wouldn't neccesarily expect them, maybe there is something more interesting to be done there.
